CSS

1. link & import
  • link : rel = “stylesheet” href = “css파일 경로”
  • import : @import url("css파일 경로");


2. 선택자(selector)
    id & class
  • id
    • 특정한 태그 1개에만 css 적용할 때 사용, 하나의 html 안에서 유일한 값임(ex.주민등록번호)
    • #으로 선언
      ex. <.style> #id이름 { 적용하고 싶은 css }
  • class
    • 비슷한 특성을 주고 싶은 태그들이 여러개일때 사용
    • .(마침표)로 선언
      ex. <.style> .class이름 {적용하고 싶은 css }
    부모 자식 선택자(tree 구조)
  • 부모: 상위에 하위 요소를 감싸고 있는 요소
  • 자식: 하위에 있는 요소


3. 속성
  • background(-color, image, repeat)
  • Typography (font style)
    • 글자과 문단의 속성을 제어, 스타일 적용
    • 스타일 종류
      추가 학습 하러가기
      1. 글자
        • color (ex. red, RGB(255,255,255), #FFFFFF)
        • font-family: 글꼴 설정
        • font-size : 글자 크기 설정
        • font-weight : 글자 굵기 설정 (bold, normal, lighter, 100~900)
        • line-height: 행간 조정
      2. 문단
        • text-align (center, left/right, justify)
        • text-indent : 들여쓰기
        • text-decoration : 글자 장식 밑줄
        • text-decoration : 대문자, 소문자 지정
